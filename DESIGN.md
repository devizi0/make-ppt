# PPT 디자인 가이드

토스(Toss) 스타일 기반 슬라이드 디자인 시스템.

---

## 핵심 원칙

1. **여백이 곧 디자인** — 콘텐츠보다 여백이 많아야 한다
2. **텍스트 위계** — 크기 차이로 정보 계층을 만든다
3. **색은 최소** — 배경 + 텍스트 + 강조색 1개
4. **이모지는 토스페이스** — 시스템 이모지 절대 사용 금지

---

## 색상 시스템

### 다크 테마 (타이틀/강조 슬라이드 전용)
```css
--bg-dark:      #0a0a0a;   /* 슬라이드 배경 */
--bg-card:      #141414;   /* 카드/섹션 배경 */
--bg-subtle:    #1c1c1c;   /* 미묘한 구분 */
--text-primary: #ffffff;
--text-secondary: #888888;
--text-muted:   #444444;
--accent:       #3182f6;   /* 토스 블루 */
--accent-light: #ebf3fe;
--border:       rgba(255,255,255,0.08);
```

### 라이트 테마 (본문 슬라이드 기본)
```css
--bg-light:     #ffffff;   /* 슬라이드 배경 */
--bg-card:      #f8f9fa;   /* 카드/섹션 배경 */
--bg-subtle:    #f2f4f6;   /* 미묘한 구분 */
--text-primary: #0a0a0a;
--text-secondary: #6b7684;
--text-muted:   #b0b8c1;
--accent:       #3182f6;   /* 토스 블루 */
--accent-light: #ebf3fe;
--border:       rgba(0,0,0,0.06);
```

### 배경 규칙 (엄수)
| 슬라이드 유형 | 허용 배경 |
|---|---|
| 타이틀 (첫 장) | `#0a0a0a` 단색 다크 |
| 마무리 (끝 장) | `#0a0a0a` 단색 다크 |
| 목차 / 본문 | `#ffffff` 완전 흰색 |
| 강조 섹션 | `#0a0a0a` 완전 검정 |
| **그라디언트** | **타이틀/마무리에만 허용, 본문 절대 금지** |

## 타이포그래피

### 폰트 스택
```css
font-family: 'Pretendard', -apple-system, 'Apple SD Gothic Neo',
             'Noto Sans KR', sans-serif;
```

### 텍스트 스케일 (1920×1080 기준)
| 용도 | 크기 | 굵기 |
|---|---|---|
| 메인 헤드라인 | 96~120px | 800 |
| 섹션 제목 | 64~80px | 700 |
| 서브 제목 | 40~52px | 600 |
| 본문 | 28~32px | 400 |
| 캡션/레이블 | 18~22px | 500 |
| 태그/배지 | 16~18px | 600 |

### 행간 / 자간
```css
/* 헤드라인 */
line-height: 1.15;
letter-spacing: -0.02em;

/* 본문 */
line-height: 1.6;
letter-spacing: -0.01em;

/* 레이블/태그 */
letter-spacing: 0.04em;
text-transform: uppercase;
```

## 이모지 — 토스페이스

### CDN 로드 (head에 반드시 포함)
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/toss/tossface/dist/tossface.css">
```

### 사용법
```html
<!-- 클래스 방식 -->
<span class="tossface">🚀</span>

<!-- CSS 적용 -->
.tossface {
  font-family: 'TossFace';
  font-size: 48px;   /* 슬라이드 콘텍스트에 맞게 조정 */
}
```

### 이모지 크기 가이드
| 용도 | 크기 |
|---|---|
| 아이콘 (카드 내) | 48~56px |
| 섹션 대표 이모지 | 72~96px |
| 타이틀 포인트 이모지 | 120~160px |

## 레이아웃 시스템

### 그리드
```css
/* 기본 여백 */
padding: 120px 140px;

/* 카드 그리드 */
display: grid;
grid-template-columns: repeat(3, 1fr);  /* 3열 */
grid-template-columns: repeat(2, 1fr);  /* 2열 */
gap: 32px;
```

### 카드 컴포넌트
```css
.card {
  background: var(--bg-card);        /* 라이트: #f8f9fa / 다크: #141414 */
  border-radius: 24px;
  padding: 48px;
  border: 1px solid var(--border);
}
```

### 구분선
```css
.divider {
  width: 64px;
  height: 4px;
  background: #3182f6;
  border-radius: 2px;
  margin-bottom: 40px;
}
```

## 슬라이드 유형별 패턴

### 타이틀 슬라이드
```
배경: #0a0a0a
┌─────────────────────────────┐
│                             │
│  [태그/날짜]                │  ← 18px, #3182f6, letter-spacing 넓게
│                             │
│  메인 헤드라인              │  ← 108px, white, bold
│  두 번째 줄                 │
│                             │
│  서브타이틀 설명 텍스트     │  ← 28px, #888
│                             │
└─────────────────────────────┘
```

### 목차 슬라이드
```
배경: #ffffff
┌─────────────────────────────┐
│  목차                       │  ← 64px, #0a0a0a
│  ────                       │  ← blue divider
│                             │
│  ┌────────┐  ┌────────┐    │
│  │ 01 항목│  │ 02 항목│    │  ← 카드 그리드
│  └────────┘  └────────┘    │
│  ┌────────┐  ┌────────┐    │
│  │ 03 항목│  │ 04 항목│    │
│  └────────┘  └────────┘    │
└─────────────────────────────┘
```

### 본문 슬라이드 (리스트형)
```
배경: #ffffff
┌─────────────────────────────┐
│  [섹션 레이블]              │  ← 18px, #3182f6
│  섹션 제목                  │  ← 72px, #0a0a0a
│  ────                       │
│                             │
│  🚀 항목 제목               │  ← tossface + 28px bold
│     설명 텍스트             │  ← 22px, #6b7684
│                             │
│  ✅ 항목 제목               │
│     설명 텍스트             │
└─────────────────────────────┘
```

```
│  │      │ │      │ │      ││
│  │ 제목 │ │ 제목 │ │ 제목 ││
│  │ 설명 │ │ 설명 │ │ 설명 ││

```

## 금지 사항

- 본문 슬라이드에 그라디언트 배경 사용
- 시스템 기본 이모지 사용 (토스페이스로 대체)
- 3가지 이상 강조색 혼용
- 슬라이드 한 장에 7줄 이상 텍스트
- 테두리/그림자 과용 (카드당 최대 1개)
- 밑줄, 이탤릭 남용
- 배경이 회색 계열 (#ffffff, #0a0a0a만 허용)
